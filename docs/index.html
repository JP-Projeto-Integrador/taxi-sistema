Sistema de Táxi

Projeto Integrador - Aplicação Java com MySQL

🧭 Sobre o Projeto

Este sistema foi desenvolvido como parte de um projeto integrador. O objetivo é gerenciar corridas de táxi, motoristas, clientes e veículos usando Java e banco de dados MySQL.

⚙️ Funcionalidades

  • Cadastro e listagem de clientes
  • Cadastro e gerenciamento de motoristas
  • Cadastro de veículos
  • Registro de chamados (corridas)
  • Interface com menus no terminal

🚀 Tecnologias Utilizadas

  • Java
  • MySQL
  • JDBC
  • Paradigma de Programação Orientada a Objetos
  • Git & GitHub
